* One-sentence summary

The harness CLI is the public protocol.  A local daemon is a private
acceleration cache.  MCP is explicitly excluded from the architecture.

This file is the stable RFC 005 entry point.  The normative body is
split into focused Org modules under =rfcs/cli-first-harness-ux/= so
agent-facing command contracts, graph/search behavior, hook guidance,
and historical readiness notes can evolve without growing one large
always-read document.
