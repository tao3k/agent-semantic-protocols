* Scope

This records the first repair-oriented real-project pass for the software criterion
signal contract.  The pass starts from the default compact output rather than
JSON, follows one =native-idiom.manual-transform-loop= signal in the TypeScript
harness, applies a small repair, and re-runs the same real-project checks.

* Repair path

The compact output exposed:

#+begin_example
Help: Public function 'ownersForHits' uses 1 manual transform loop(s). Signals: native-idiom.manual-transform-loop.
#+end_example

Following that signal led to =ownersForHits= in
=src/cli/semantic-search/owners.ts=.  The function manually built a return
collection using =seen=, =owners.push(...)=, and =return owners=.  The nearby
=uniqueOwners= helper used the same shape.  The repair introduced a local
=uniqueBy= helper and rewrote the public helpers to compose =uniqueBy(...)= with
=map= or =flatMap=.

The repaired code preserves stable owner order and de-duplicates by the same
path key.  The signal disappeared for =ownersForHits= and =uniqueOwners=, while
the remaining four TypeScript manual-transform-loop findings stayed localized
to source-text and native-syntax fact collection functions.
