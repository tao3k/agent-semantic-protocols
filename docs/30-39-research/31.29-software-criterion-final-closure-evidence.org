* Final fixes after clean evidence

Two final closure issues were found after the Python clean evidence pass.

** Package-local schema profile drift

The shared schema sync gate rejected generated
=semantic-fact-ontology.v1.schema.json= copies in TypeScript and Julia provider
schema directories.  Those copies were removed from the nested provider indexes
and worktrees.  The root shared schema remains the source of truth.

** Rust full-test compile drift

The Rust changed check passed, but the focused Rust software-criterion snapshot test
compiled a wider surface and found a stale CLI model drift:
=QueryOptions= parsed =--workspace= into =workspace_root=, while the struct had
no field.  The fix added =QueryOptions.workspace_root= and removed the unused
=QuerySearchOptions.workspace_root= field to satisfy =#![deny(dead_code)]=.
